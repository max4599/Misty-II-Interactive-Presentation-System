# Misty-II-Interactive-Presentation-System
Entwicklung einer interaktiven Präsentation für Besucher mit Sprachausgabe, Bewegungssteuerung und Gesichtsausdrücken

Interactive presentation system for the Misty II social robot.

Features
	•	Voice output and audio effects
	•	Facial expressions and LED control
	•	Head and arm movements
	•	Automated presentation sequence
	•	Sleep and wake-up behaviour

Technologies
	•	Python
	•	Misty II API

Description

This project implements an interactive presentation for visitors using the Misty II robot. The robot welcomes guests, performs predefined movements, uses facial expressions and speech output, and interacts with the audience during a presentation.
